# VA Pre-Ban Mag Buy List

**Live report → [hazyicestudios.github.io/VA-PreBan-MagBuyList](https://hazyicestudios.github.io/VA-PreBan-MagBuyList/)**

Interactive purchasing guide for stocking up on high-capacity magazines and pistols before Virginia's upcoming ban takes effect on **July 1, 2026**.

---

## What it does

- **34 pistols** ranked by community popularity, covering 9mm, 10mm, .22 LR, .22 WMR, .380 ACP, and .45 ACP
- Per-capacity **Buy Mags** links (Google search) and per-model **Buy Pistol** links (GunBroker) on every row
- Live **countdown** to the July 1, 2026 deadline
- **Pre-ban flag** on every entry — instantly see which guns need action now vs. which are already covered or under the limit
- **Filters** by pre-ban status, caliber, purpose, and max gun price
- **Source filter** to toggle between spreadsheet data and additional community recommendations
- Sortable columns, estimated mag prices, full compatibility lists (compatible pistols + PCCs)
- Includes a dedicated **10mm / wilderness / bear defense** section

## Filters

| Filter | Options |
|--------|---------|
| Pre-Ban Action | All / Needed / Not Needed |
| Caliber | All / 9mm / 10mm / .22 / .380 / .45 ACP |
| Purpose | EDC, Home Defense, Competition, Wilderness, etc. |
| Max Gun Price | Any / $350 / $500 / $700 / $1,000 |
| Source | All / Spreadsheet / My Picks |

## Legal context

Virginia legislation banning the **sale** of magazines capable of holding more than 15 rounds takes effect July 1, 2026. Items purchased before the effective date are **grandfathered** — this guide helps identify what to buy before the deadline.

> This guide is for informational purposes. Verify current Virginia law before making purchases.
